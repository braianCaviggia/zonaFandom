Acerca del Proyecto

Esta aplicación web fue desarrollada como un proyecto de práctica con el objetivo de gestionar salas temáticas de forma interactiva.

El usuario puede elegir entre distintas salas (Series, Deportes y Videojuegos) y acceder a contenido relacionado en cada una.

La aplicación está construida con HTML, CSS y TypeScript, ofreciendo una experiencia visual simple, clara y dinámica.
